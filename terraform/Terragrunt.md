#### Install Terragrunt:
```bash
curl -L "https://github.com/gruntwork-io/terragrunt/releases/download/v${version}/terragrunt_linux_amd64" -o terragrunt && \
chmod +x ./terragrunt && mv ./terragrunt /usr/local/bin/
```
